Seam Icon Generator

Instructions:

  1. Open this file in Chrome/Firefox
  2. Right-click each icon and select "Save image as..."
  3. Save as icon16.png, icon48.png, and icon128.png in the icons folder
  4. Update manifest.json to include the icons section (see below)

Add this to your manifest.json:

"icons": {
  "16": "icons/icon16.png",
  "48": "icons/icon48.png",
  "128": "icons/icon128.png"
}
